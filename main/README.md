# UNITEN CareSpace

UNITEN CareSpace is a mental health support web platform developed as a Final Year Project (FYP).

## Features
- Student counseling appointment system
- Counselor dashboard and session management
- Admin monitoring and management
- Secure role-based access (Admin / Counselor / Student)

## Tech Stack
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP

## Notes
- Database files are provided in `.sql` format
- `config.php` is excluded for security reasons
